# Sistema de Login e Cadastro de Usuários usando Python e SQL
Este é um repositório que contém um sistema de login e cadastro de usuários desenvolvido em Python e SQL. O objetivo deste projeto é demonstrar como é possível criar um sistema de autenticação básico utilizando essas tecnologias.

## Funcionalidades
O sistema possui as seguintes funcionalidades:

- Cadastro de novos usuários com nome, email, usuário e senha
- Autenticação de usuários existentes

## Requisitos
Para executar o projeto, é necessário ter os seguintes requisitos instalados:

- Python 3.x
- MySQL ou outro sistema de gerenciamento de banco de dados compatível com SQL

## Como executar o projeto
1) Clone este repositório em sua máquina
2) Importe o arquivo do banco de dados "UserData.db"
3) Execute o banco e o arquivo em python "DataBaser.py".
Acesse o sistema no seu navegador em http://localhost:5000.
